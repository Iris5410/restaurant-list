# 餐廳清單
一個能紀錄、收藏喜愛餐廳的清單網站。  

## 功能
1.可以新增餐廳    
2.可以瀏覽所有餐廳    
3.可以瀏覽一家餐廳    
4.可以編輯餐廳資訊    
5.可以刪除餐廳    

## 專案安裝流程
1.將此專案 clone 到本地    
2.進入專案資料夾後，安裝 npm：  
``` npm install ```    
3.接著安裝 nodemon：  
``` nodemon install ```    
4.匯入種子檔案：  
``` npm run seed ```    
5.當檔案匯入成功，終端機會顯示：done    
6.最後啟動伺服器：  
``` npm run dev ```    
有看見 Express is listening on localhost:3000，就可以打開 http://localhost:3000/ 使用餐廳清單囉
